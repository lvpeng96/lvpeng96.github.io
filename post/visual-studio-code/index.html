C++环境配置
1.下载mingw 选择win版本安装
2.经mingw路径的/bin路径添加到系统内的环境变量中（测试是否成功方法命令行中输入gcc -v）
3.在VScode运行 选择C++(GDB/LLDB) 然后选择gcc.exe。
